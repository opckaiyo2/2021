## 役立つ知識
***

pythonでは複数行コメントアウトするときに\"""で囲います  
下のように記述すると初めの\"""の前#を切り替えることで簡単に複数行コメントを切り替えられます
> 例  
> この時はprint文は実行される
>> \#"""  
>> print("test")  
>> \#"""
>
> この時はprint文は実行されない
>> \#"""  
>> print("test")  
>> \#"""
